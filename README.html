SHNU-Exam

上海师范大学考试卷 LaTeX 文档类。精确还原上海师范大学标准试卷格式，包括字体、字号、行间距、得分表和答题区。

快速开始

\documentclass{shnu-exam}

\examtitle{概率论与数理统计}
\examtype{A卷}
\semester{2025~2026 学年~~第~1~学期}
\examdate{2026年1月13日}
\examduration{90}

\begin{document}
\maketitle

\section{一、选择题（每小题2分，共16分）}

\question 题干内容...
\choice{选项A}
\choice{选项B}
\choice{选项C}
\choice{选项D}

\section{二、填空题（每空3分，共24分）}

\question 题干 \fillin

\section{三、计算题（本题10分）}

\problem[lines=15]{题干内容...}

\end{document}

编译要求

  • 编译器：XeLaTeX（必须，需要中文字体支持）
  • 编译次数：2 次（首次生成 .aux，第二次得分表完整显示）
  • TeX 发行版：TeX Live 2024+（含 ctex、fontspec、unicode-math 等宏包）

系统字体要求

字体用途Windows 自带
宋体 SimSun正文汉字✓
黑体 SimHei标题、大题标题✓
Times New Roman正文西文、数学✓
Tex Gyre Termes Math数学公式TeX Live 自带

文档类选项

\documentclass[
  answerspace=10,    % 大题默认答题空行数（默认 10）
  showscore=true,    % 是否显示得分表和得分栏（默认 true）
  solution=false,    % 是否显示答案解析（默认 false）
]{shnu-exam}

命令参考

试卷信息

命令说明
\examtitle{科目名称}科目名称，显示在"科目："后
\examtype{A卷}卷型
\semester{学年~~学期}学年学期信息
\examdate{日期}考试日期
\examduration{分钟数}考试时长
\maketitle生成试卷头 + 得分表 + 诚信承诺行

大题

命令说明
\section{标题}大题标题（自动计入得分表，自动在标题旁加"得分"栏）

选择题

每道题包含题干（\question）和四个选项（\choice）。选项排版根据长度自动选择：

  • 4 列一行：四个选项都较短时，排在一行
  • 2×2 网格：每个选项不超过半行宽，但四个放不下时
  • 4 行堆叠：选项较长时，每个一行

填空题

命令说明
\fillin生成空白横线（3cm）
\fillin[答案]当 solution=true 时显示红色答案

解答题答题区

命令说明
\problem{题干}使用全局默认行数（answerspace 选项）
\problem[lines=15]{题干}预留 15 行空白答题线
\problem[space=6cm]{题干}预留 6cm 空白区域

注意事项

编译顺序：首次编译时得分表仅显示"题号 | 总分"两列（因为 .aux 文件尚未生成）。第二次编译将显示完整的大题列（一、二、三…）。此行为与 LaTeX 目录生成一致。
